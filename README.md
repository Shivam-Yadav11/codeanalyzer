A smart and efficient Code Analyzer that helps developers understand their code better. Simply paste your code into the analyzer, and it instantly provides a complete breakdown of your program structure and performance.

The analyzer detects:

⏱️ Time Complexity
💾 Space Complexity
🧩 Variables and Variable Names
🔁 Loops used in the code
⚙️ Functions and Function Names
🪙 Total Number of Tokens

This project is useful for developers, students, and programmers who want to analyze, optimize, and better understand their code quickly and efficiently.
